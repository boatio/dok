그럼 인제 터미널에 명령어를 적어보도록 하겠습니다<br>
`dok html -y` 이렇게 dok 터미널에 적어주시고 엔터를 눌려주세요<br>
그러면 자동 css , html 파일을 포함하고 있는 dok<br>
예제 파일이 만들어졌습니다.<br>
`-y`는 html dok 터미널에서 html 예제 파일을 <br>
만든 다는 뜻입니다<br>
그럼 다시 디렉토리를 보면
<pre><code>
|─현재 디렉토리

</code></pre>
